Welcome to the Student API

This project provides a RESTful API to manage student records, including their name and passport number. You can view all students and update their details through easy-to-use endpoints. The API is built with Spring Boot, JPA, an in-memory H2 database, and includes Swagger UI for documentation and testing.

This is a demo backend powered by Spring Boot, JPA, H2 DB, and Swagger UI.
